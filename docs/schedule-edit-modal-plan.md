# Schedule Edit Modal Plan

## Goal

Rework the character schedule editor so it feels like part of the redesigned conversation status and schedule surfaces, while staying easy to scan on desktop and touch-friendly on mobile.

## Incremental Steps

- [x] Inspect the current schedule card, modal shell, and shared neutral panel styles.
- [x] Rebuild the modal content into a clearer summary-first layout with better grouping.
- [x] Tighten the day editor affordances for faster scanning, editing, and deletion.
- [x] Remove extra chrome that made the modal feel like a separate surface family.
- [ ] Verify the layout on smaller screens and adjust spacing or width if it feels cramped.
- [ ] Run the client checks and fix any regressions introduced by the redesign.

## Notes

- Keep the modal aligned with the conversation presence surfaces instead of introducing a new visual language.
- Prefer compact, readable controls over decorative chrome.
- Preserve the existing schedule editing behavior, only improve the presentation and interaction flow.
